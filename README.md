# PALUF Control 2026

Sistema de control de funciones para PALUF Ecoaldea.

## 🌿 Sobre PALUF

PALUF es un centro holístico y reserva forestal ubicado en Tenjo, Cundinamarca, Colombia.

## 📱 Funcionalidades

- ✅ 152 funciones pre-cargadas
- ✅ Control de asistencia diario (✓ ✗ -)
- ✅ Dashboard con ranking de desempeño
- ✅ Sistema de comentarios
- ✅ CRUD de tareas
- ✅ Filtros por frecuencia
- ✅ Navegación por meses
- ✅ Responsive (móvil, tablet, PC)
- ✅ Marcado rápido
- ✅ Impresión
